# nsbtools
Neuroimaging tools developed by the Neural Systems and Behaviour Lab.

## Installation
`nsbtools` works with Python 3.9 and above. It can be installed by cloning the repository and installing from the local directory.

```
git clone https://github.com/NSBLab/nsbtools.git
cd nsbtools
pip install .
```
