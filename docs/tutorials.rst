.. _tutorials:

----------
Tutorials
----------

These tutorials cover the main features of ``nsbtools``, including computing geometric eigenmodes, 
 decomposing and reconstructing cortical maps, simulating cortical activity, generating structural 
 connectomes, and visualising cortical maps. For further details, please refer to the API 
 documentation. If you encounter any issues, please open an issue on the `GitHub repository 
 <https://github.com/NSBLab/nsbtools/issues>`_.
